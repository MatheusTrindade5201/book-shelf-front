A pasta utils se destina para os utilitários da aplicação, inicialmente separei entre:
- Constants:
    Aqui guardaremos variaveis constantes que podem ser usadas globalmente na aplicação.

- Helpers: 
    Aqui guardaremos as funções auxiliares que utilizaremos na aplicação.

- Schema:
    Aqui guardaremos caso necessário, modelos de objetos que podemos usar na aplicação.
